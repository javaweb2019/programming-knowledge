# 元素大小和位置
- client
  - clientWidth,clentHeight
    元素的可视化宽高(不包含边框)
  - clientLeft,clientTop
    元素的左上边框宽度
- offset
  - offsetWidth offsetHeight
    - 元素的宽高(包含边框)
  - offsetLeft,offsetTop
    - 距离最近的定位元素的左或上距离
- scroll
  - scrollWidth,scrollHeight
    - 元素的滚动宽度和高度(所有宽度和高度)
  - scrollLeft,scrollTop
    - 滚动条滚动的距离
